Layer 2 Prompt Template
I already have a complete HTML skeleton for [business name],
live at [your GitHub Pages URL].

Please add a style.css file that applies the following
design system. Do not change the HTML structure.

Do not add responsiveness yet. Do not add hover states yet.
Colour, typography, and spacing only.

COLOUR — as CSS custom properties in :root, not hard-coded:
--color-primary: [HEX]   --color-secondary: [HEX]
--color-bg: [HEX]      --color-text: [HEX]
--color-nav-bg: [HEX]   --color-cta: [HEX]
--color-accent: [HEX]

TYPOGRAPHY:
Heading font: [font], H1 [X]px, H2 [X]px
Body font: [font], [X]px, line-height [X]

SPACING — 8px scale, as CSS custom properties:
--space-xs: 4px  --space-sm: 8px  --space-md: 16px
--space-lg: 24px --space-xl: 32px --space-2xl: 48px --space-3xl: 64px

Apply these to: header/nav background and padding, hero
section background and heading, button background and
padding, section padding, and card/content gaps.

Step 8 — Upload
  1. Go to your repository on github.com
  2. Add file → Upload files → drag in style.css
  3. Commit message: Add Layer 2 CSS styling
  4. Wait 30–60 seconds for GitHub Pages to rebuild
  5. Hard refresh your live URL (Ctrl/Cmd + Shift + R)
Cache warning: if your styling doesn't appear, it's almost always a caching issue, not a broken file. Hard refresh before you start debugging.
